# PetLoris

Sistema Web desenvolvido para a disciplina **Fundamentos de Sistemas Web**.

## Projeto
O PetLoris é um site básico de um petshop fictício, criado utilizando apenas HTML, sem CSS ou JavaScript (fase 1 do projeto).

## Funcionalidades
- Página inicial com apresentação da loja  
- Página de produtos com imagens, descrições e valores  
- Página de serviços com lista e preços  
- Página de contato com informações da empresa
